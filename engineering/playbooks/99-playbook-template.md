# Template de Playbook - Base+

Use este modelo para criar novos playbooks.

## Objetivo

Descreva o objetivo do playbook e qual tipo de conversa ele deve conduzir.

## Quando utilizar

Explique em quais situacoes este playbook deve ser usado.

## Pre-requisitos

Liste documentos, contexto, arquivos ou decisoes que devem ser conhecidos antes da conversa.

Exemplo:

- `VISION.md`
- `engineering/architecture/principles.md`
- `engineering/architecture/decisions.md`
- `MODULE_TEMPLATE.md`

## Fluxo da conversa

Defina as etapas da conversa.

Exemplo:

1. Contexto.
2. Problema.
3. Alternativas.
4. Riscos.
5. Decisao.
6. Prompt.
7. Validacao.

## Perguntas obrigatorias

Liste perguntas que devem ser respondidas antes de qualquer implementacao.

Exemplo:

- Qual problema real esta sendo resolvido?
- Existe padrao Base+ aplicavel?
- Quais camadas serao afetadas?
- Ha impacto em seguranca?
- Ha impacto em operacao?
- Ha necessidade de documentacao?

## Criterios de decisao

Defina os criterios que orientam a escolha.

Exemplo:

- simplicidade;
- baixo acoplamento;
- compatibilidade;
- manutencao;
- seguranca;
- operacao;
- aderencia aos principios Base+.

## Criterios de encerramento

Defina quando a conversa pode ser encerrada.

Exemplo:

- decisao consolidada;
- riscos identificados;
- prompt gerado;
- validacoes definidas;
- proximos passos claros.

## Resultado esperado

Descreva o resultado que o playbook deve produzir.

Exemplo:

- decisao tecnica;
- prompt de implementacao;
- checklist de validacao;
- recomendacao de nao implementar, quando aplicavel.

## Proximo Playbook

Indique qual playbook deve ser usado em seguida, se houver continuidade.

Exemplo:

- playbook de implementacao;
- playbook de homologacao;
- playbook de release;
- playbook de troubleshooting.

## Checklist Final

- [ ] Objetivo entendido.
- [ ] Contexto coletado.
- [ ] Perguntas obrigatorias respondidas.
- [ ] Alternativas avaliadas.
- [ ] Riscos identificados.
- [ ] Decisao consolidada.
- [ ] Prompt gerado quando aplicavel.
- [ ] Validacao definida.
- [ ] Proximo passo indicado.
