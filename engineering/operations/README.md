# Runbooks Operacionais - Base+

Esta pasta concentra os runbooks operacionais oficiais da Base+.

Runbooks devem ser utilizados para instalar, atualizar, manter, diagnosticar, restaurar e operar ambientes baseados na plataforma.

Eles representam documentacao operacional oficial e devem ser mantidos atualizados a cada release.

Sempre que existir documentacao equivalente em `docs/`, este diretorio deve atuar como ponto central de organizacao e referencia, evitando duplicacao desnecessaria.

## Indice

- `installation.md`: instalacao em DEV, HOM e PRD.
- `update.md`: atualizacao de ambientes existentes.
- `backup.md`: backup de banco, uploads e arquivos relevantes.
- `restore.md`: restauracao de banco e uploads.
- `rollback.md`: retorno para versao anterior em caso de falha.
- `troubleshooting.md`: indice operacional de problemas conhecidos.
- `monitoring.md`: verificacoes de saude, logs, containers, volumes e banco.
- `maintenance.md`: atividades periodicas de manutencao.
